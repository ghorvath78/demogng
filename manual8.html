General Settings

A number of normally not often modified settings can be accessed via the "General Settings" window available via typing "p" (for "parameters") in the canvas or via the cog button on the interface.

A minimal explanation appears as tooltip if you hover over the parameter with the mouse.

One perhaps more interesting parameter is "initFromPD" (default: true): If this is unchecked then the intial node positions are drawn at random from the unit square (ignoring the distribution).
